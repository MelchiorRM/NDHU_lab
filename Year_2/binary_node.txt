Convert a general tree to a binary tree using the left-child/right-sibling representation.

Input format:
- First line: N (number of nodes)
- Second line: root id
- Next N-1 lines: each contains two integers `parent child` describing an edge in the general tree.

Output:
Print the resulting binary tree in a preorder traversal. Each node id should be printed on its own line, with indentation showing depth (children indented one space per level). The left pointer represents the first child and the right pointer represents the next sibling.

Example:
```
Input:
5
1
1 2
1 3
2 4
2 5

Output:
1
 2
  4
  5
 3
```
